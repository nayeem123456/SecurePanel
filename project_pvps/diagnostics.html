⚠️ If Tests Fail:

  1. XAMPP Not Running: Open XAMPP Control Panel and start Apache and MySQL services
  2. Database Not Created: Open phpMyAdmin (http://localhost/phpmyadmin) and create database 'pvps_db'
  3. Tables Missing: Import the database schema from backend/database/ folder
  4. Port Conflicts: Make sure no other services are using ports 80 (Apache) or 3306 (MySQL)
  5. File Permissions: Ensure backend files have proper read permissions
